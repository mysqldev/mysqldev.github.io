MySQL历史与版本

2025-08-17

MySQL历史

MySQL最初为MySQL AB公司开发，该公司于2008年被Sun Microsystem收购，2009年Oracle收购了Sun Microsystem。MySQL目前由Oracle公司研发团队作为官方团队开发并发行。
MySQL可能是最流行的关系数据库，尤其是在互联网企业中应用很广。开源建站软件套件LAMP中的M即MySQL。

MySQL版本

Oracle官方版本

MySQL社区版

开源社区免费版本，源码见 https://github.com/mysql/mysql-server

MySQL企业版

Oracle提供的付费企业版，支持透明加密、备份、高可用、审计、监控、Workbench、技术支持服务等企业级特性。

MySQL Cluster

主打扩展性，支持多副本、分布式查询、内存型实时数据库、自动分区等。

MySQL Heatwave

融合OLTP & OLAP & ML的一体化解决方案，可在OCI(Oracle Cloud Infrastrucutre)、AWS、Azure云上部署。

Percona Server

MySQL的一个知名的发行版，由原MySQL测试团队主导推出，免费提供了一些MySQL企业版的付费特性（如支持线程池，支持RocksDB存储引擎），以及丰富的生态运维工具如xtrabackup、percona-toolkit等。

MariaDB

MySQL被Oracle收购后，原团队Monty等人担心MySQL的开放性，因此推出了MariaDB。MariaDB也支持了MySQL企业版的一些企业级特性如线程池、审计等功能。MariaDB最初目标是兼容MySQL数据格式，但随着各自项目的演化，逐渐无法继续兼容。
